# Movie Searcher Compose
Movie Searcher Compose is an Android app built with Jetpack Compose.

## Tech stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Jetpack Navigation
- Coroutines
- StateFlow
- Hilt - Dagger
- Retrofit
- Accompanist
- OMDb Api
